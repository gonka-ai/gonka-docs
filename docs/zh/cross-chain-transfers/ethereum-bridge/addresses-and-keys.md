# 地址与密钥

这是在跨链操作前最需要理解的关键页面。**请在首次转账前仔细阅读。**

## 一把密钥，两个地址

以太坊（Ethereum）和 Gonka 均使用相同类型的加密密钥（一个 `secp256k1` 密钥对）。因此，单个私钥同时控制着**两条链**上的账户。两者唯一的区别在于如何将公钥转换为人类可读的地址：

| 链 | 地址格式 | 地址从公钥推导的方式 |
| --- | --- | --- |
| 以太坊 | `0x...` |  |
| （20 字节，十六进制） |  |  |
因此，同一个私钥会生成**两个外观不同的地址** —— 一个是 `0x…`，另一个是 `gonka1…` —— 但这两个地址都由同一把私钥控制。

## 跨链桥如何决定你的代币去向

当你将代币从以太坊跨链到 Gonka 时，你需要将代币发送至桥接合约，并使用你的以太坊私钥签署这笔交易。Gonka 跨链桥会：

1. 检测以太坊上已确认的存款交易；
2. **从交易签名中恢复公钥**；
3. 使用该公钥计算出对应的 **Gonka 地址**（即上述 `gonka1…` 标准推导方式）；
4. 将跨链代币铸造或释放到该 `gonka1…` 地址。

换句话说：**被封装的代币会被发送到由同一公钥生成的 Gonka 地址 —— 该地址由签署以太坊存款交易的私钥所控制。** 要使用这些代币，你必须在 Gonka 链上使用相同的私钥。

反向操作则有所不同：当你从 Gonka **跨链回以太坊**时，你可以在取款交易中**显式指定目标地址**；而当你**跨链到 Gonka 时无法选择接收地址** —— 它完全由你的密钥决定。

## 助记词陷阱（请务必阅读！）

大多数用户从未直接看到自己的原始私钥 —— 他们只拥有一个**助记词**（mnemonic），并由钱包自动从中派生密钥。这虽然方便，但在跨链时容易造成误解：

一个助记词并不对应单一密钥。钱包通过 **BIP-44 派生路径**从助记词生成密钥，而每条区块链使用**不同的路径**：

* 以太坊钱包使用币种类型 **60** → 路径为 `m/44'/60'/0'/0/0`
* Cosmos/Gonka 钱包使用币种类型 **118** → 路径为 `m/44'/118'/0'/0/0`

由于路径不同，**同一个助记词会为以太坊和 Gonka 生成完全不同的私钥**，从而产生两个毫无关联的地址。如果你从一个由助记词派生的以太坊账户发起存款，然后在钱包中查看由助记词派生的 Gonka 账户，**这两个账户并非同一把密钥**，因此跨链的代币将被发送到一个你的钱包并未显示、甚至可能无法控制的 `gonka1…` 地址。

!!! danger
    不要假设“相同的助记词 = 两条链上的同一账户”。跨链桥需要的是**两条链上使用相同的私钥**，而不是相同的助记词。若使用标准但不同的派生路径，资金将被发送到基于你以太坊私钥推导出的 Gonka 地址 —— 而你的 Gonka 钱包因使用不同路径派生，无法控制该地址。这并非永久性损失，你仍可从同一助记词中派生出原始以太坊私钥，该私钥即可控制 Gonka 链上的接收账户。

## 如何获取匹配的 Gonka 地址

你有两种选择。

### 选项 A — 使用仪表盘（推荐）

Gonka 仪表盘可自动解决密钥派生问题。只需使用你用于跨链的**同一个以太坊钱包**连接仪表盘，系统便会自动推导并显示跨链桥将使用的正确 `gonka1…` 地址，展示你的封装资产余额，并引导你完成存款/取款流程。这种方式无需手动处理原始私钥，也能避免上述助记词带来的混淆问题。

访问仪表盘地址：

```text
https://node1.gonka.ai:8443/dashboard/
```

### 选项 B — 将相同的私钥导入 Gonka 密钥环

如果你使用命令行操作，请将控制你以太坊账户的**完全相同**的 `secp256k1` 私钥（十六进制）导入到 Gonka 密钥环中。生成的 `gonka1…` 地址即为跨链桥铸造资产的目标地址：

```bash
inferenced keys import-hex <key_name> <YOUR_PRIVATE_KEY_HEX>

# Show the derived Gonka address
inferenced keys show <key_name> -a
```

此处打印的地址正是将接收您跨链代币的 `gonka1…` 地址，且该密钥可为这些代币的 Gonka 交易（转账、提现）进行签名。

!!! warning
    导入原始私钥会将其暴露于您导入的设备和密钥环中。建议在安全设备上使用基于文件的密钥环（`--keyring-backend file`），切勿在不受信任的主机上粘贴保护大量以太坊资金的私钥。如有疑问，请使用仪表板。
