# 通过仪表板（UI 指南）进行跨链桥接

仪表板是进行跨链桥接最简单的方式。它会自动为您处理密钥派生（参见[地址与密钥](addresses-and-keys.md)），因此您无需手动导入私钥或计算地址。您可以在以下地址打开仪表板：

```text
https://node1.gonka.ai:8443/dashboard/
```

## 仪表板能为你做什么

* **推导出正确的地址。** 连接你用于跨链的以太坊钱包后，仪表板会显示对应的 `gonka1…` 地址，即跨链桥将代币发送到的目标地址——这样你始终清楚你的封装代币将到达何处。
* **警告助记词账户风险。** 如果你使用的是其以太坊和 Gonka 密钥源自同一**助记词**的钱包，仪表板会检测到并发出警告，因为由种子派生的账户在不同链上使用不同的密钥，可能导致资金被发送到你无法控制的地址。详见 [地址与密钥](addresses-and-keys.md) 获取完整说明。
* **显示你的跨链余额**，在“跨链资产”部分展示，以便你确认存款是否已到账。
* **报告链的运行状态**，让你在发起转账前可以查看链是否处于降级状态。
* **提示更新跨链桥纪元**，当以太坊跨链桥落后于 Gonka 链时。如果你看到 **跨链桥需要更新纪元** 的提示，请点击 **更新跨链桥** 以提交缺失的纪元密钥。这是一笔普通的以太坊交易，因此由连接的钱包支付 gas 费。详见 [跨链桥纪元更新](bridge-epoch-update.md)。

## 支持的操作流程

仪表板支持全套跨链操作，无需使用命令行工具（CLI）：

* **以太坊 → Gonka**：存入任意 ERC-20 代币（如 USDT、USDC、WETH 等），并在 Gonka 上获得对应的封装代币。
* **Gonka → 以太坊**：将封装代币提现回指定的以太坊地址。
* **GNK ↔ WGNK**：将原生 GNK 跨链至以太坊作为 WGNK，或反向操作。

## 典型存款流程（以太坊 → Gonka）

1. 打开仪表板并连接你的以太坊钱包。
2. 确认仪表板显示的 `gonka1…` 收款地址是你所控制的地址。
3. 选择代币和金额，并授权将代币转账给跨链桥合约。
4. 等待约 15–20 分钟以等待以太坊最终确认（参见 [时间与最终性](overview.md#timing-finalization)），然后在“跨链资产”部分查看你的封装代币余额。

## 典型提现流程（Gonka → 以太坊）

1. 连接钱包，选择要提现的封装代币及数量，并输入目标 **以太坊** 地址。
2. 确认提现操作。仪表板将收集当前纪元的 BLS 签名，并向以太坊上的跨链桥合约提交释放交易。

!!! 提示
    如果你不熟悉命令行工具（CLI）或不希望手动处理私钥，推荐使用仪表板。如需手动操作，CLI 的分步流程文档请参见 [存入 USDT](deposit-usdt.md)、[提现 USDT](withdraw-usdt.md)、[存入 GNK](deposit-gnk.md) 和 [提现 GNK](withdraw-gnk.md)。
